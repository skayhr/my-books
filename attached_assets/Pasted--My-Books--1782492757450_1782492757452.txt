وثيقة المواصفات الفنية والمشروع: تطبيق "My Books"
النظام: نظام أرشفة وإدارة الخطابات الرسمية والإجازات للموارد البشرية (HR).

المنصات المستهدفة: متقاطع المنصات (Desktop, Web, iOS, Android).

1. محددات العرض المستندة إلى نوع الجهاز (Responsive Layout Rules)
يجب على النظام التكيف برمجياً وتغيير واجهة المستخدم بناءً على نوع المنصة أو خيار المستخدم في شاشة الدخول (image_e746da.png):

عرض أجهزة الكمبيوتر (Desktop / Web / PC Mode):

يتم عرض التطبيق بكامل خصائصه ووظائفه وشريط التنقل العلوي بدون أي قيود.

عرض الهواتف الذكية (Mobile / Phone Mode):

يتم تقييد واجهة المستخدم ليظهر شريط البحث فقط لإجراء عمليات البحث السريع واستعراض المستندات، مع إخفاء بقية الميزات الإدارية المتقدمة لتبسيط الاستخدام أثناء التنقل.

2. المواصفات العامة والتصميم (UI/UX)
الهوية البصرية: تصميم عصري مريح للعين، زوايا مستديرة للعناصر، حقول إدخال نظيفة (Clean Input Fields)، مع تأثيرات بصرية (Animations) سلسة عند الانتقال والضغط.

الدعم ثنائي اللغة (Arabic/English): واجهة مستخدم كاملة تدعم اللغتين. تبديل فوري وسلس دون إعادة تشغيل التطبيق.

ثبات شريط التنقل (Persistent Navigation Bar):

عند تغيير لغة التطبيق، لا يتغير اتجاه الشريط ولا ينعكس اتجاهه برمجياً؛ يبقى كما هو تماماً، وفقط نصوص الخدمات الإدارية هي التي تترجم.

الوضع الداكن والفاتح (Dark & Light Mode): دعم كامل للوضعين الصباحي والليلي (image_e746fb.png).

شعار التطبيق الذكي (App Logos):

شعار الواجهات الرئيسية: مسار الصورة الثابت هو E:\mybooks\logo.

شعار التقارير الرسمية: مسار الصورة الثابت هو E:\mybooks\logo for Report letters.

3. نظام إدارة الدخول والأمان (Authentication)
شاشة الدخول المستهدفة (image_e746da.png): تحتوي على خيار لتحديد نوع العرض الافتراضي (PHONE / PC)، حقول الإدخال (USERNAME / PASSWORD)، زر (Login)، وروابط (Forgot password / Sign UP).
أمان الحسابات: التحقق عبر البريد الإلكتروني وكلمة المرور لضمان خصوصية خطابات الموظفين.
إعادة تعيين كلمة المرور: إرسال رابط آمن أو رمز تحقق (OTP) إلى بريد المستخدم لإنشاء كلمة مرور جديدة فوراً.
4. نظام الفهرسة التلقائي (Dynamic Box File System)
يتم تصنيف وفهرسة الخطابات تلقائياً بناءً على ترميز أبجدي إنجليزي يمثل نوع الإجازة، وتظهر على شكل بطاقات مربعة جذابة في قسم الخزانة (image_e74623.png و image_e74646.png):

5. واجهات وخصائص شريط التنقل العلوي الثابت (PC View)
أولاً: الشاشة الرئيسية (Home) - image_e746fb.png
تعرض رسالة ترحيبية "Hello HR... Tell me what you're looking for".
شريط بحث مركزي للبحث السريع.
بطاقات إحصائية رقمية في الأسفل: (TOTAL EMPLOYEE / TOTAL TYPES / DEPARTMENTS / TOTAL LETTERS).
ثانياً: إضافة كتاب جديد (New Book) - image_e746fa.png
استمارة البيانات: (Letter Type قائمة منسدلة تربط الحرف بالبوكس تلقائياً، Employee ID، Employee Full Name، Job Title، Department، Book Date بمستعرض ذكي).
خاصية الإكمال الذكي (Excel Auto-Fill): عند إدخال الرقم الوظيفي (Employee ID) أو الاسم، يقوم التطبيق بالاستعلام فوراً من جدول الـ Excel المستورد مسبقاً في نافذة (Emp data)، وتعبئة حقول الاسم، القسم، والمسمى الوظيفي تلقائياً دون الحاجة لكتابتها يدوياً.
مرفق المستند: منطقة رفع مخصصة تدعم صيغة PDF حصراً، وزر حفظ سحابي منظم.
ثالثاً: خزانة الكتب (Bookcase) - image_e74623.png / image_e746a3.png
تظهر بطاقات المجلدات (Box Files). عند النقر على أي مجلد (مثلاً المجلد A)، تفتح نافذة تحتوي على محرك بحث خاص بهذا المجلد فقط (image_e746a3.png).
البحث السريع للمعاينة: بمجرد كتابة اسم الموظف تظهر النتائج فورا كأشرطة بيانات تفصيلية. عند النقر للمعاينة، يفتح مستعرض PDF داخلي مدمج لعرض المجلد كاملاً مع أزرار طباعة مباشرة عبر الـ Wi-Fi أو التصدير.
رابعاً: نظام التقارير (Report) - image_e74673.png / image_e7473b.png
منطق العمل: عند كتابة اسم موظف معين في شريط بحث التقارير، يقوم النظام بتجميع كافة الخطابات والكتب الخاصة به وعرضها على شكل جدول أشرطة (كل شريط يحتوي على تفاصيل الخطاب الإدارية فقط دون عرض نسخة الكتاب).
تصدير التقرير والطباعة (image_e7473b.png): نافذة منبثقة (Pop-up) منسقة وجاهزة للطباعة تحتوي على:
شعار التقرير في الأعلى جهة اليسار من المسار المحدد (logo for Report letters).
عنوان التقرير واسم الموظف بوضوح في المنتصف.
جدول منظم يحتوي الحقول: (ID / EMPLOYEE / JOB TITLE / DEPARTMENT / LETTERS TYPE / REQUEST / DATE).
في الأسفل جهة اليسار: مساحة مخصصة لتوقيع الموارد البشرية (Human Resources Signature) لتوثيق التقرير، وزر (Print / Cancel).
خامساً: إدارة بيانات الموظفين (Emp data) - image_e746fb.png / image_e746fb.png (مستوحاة من أخر صفحة موصوفة)
شاشة تحتوي على خيار (Import Excel) لاستيراد ملف البيانات الأساسي للموظفين من الكمبيوتر وتخزينه في قاعدة البيانات المحلية للتطبيق لتشغيل ميزة الإكمال التلقائي الذكي عبر الـ ID.
جدول يعرض: (No. / Employee ID / Employee Full Name / Job Title / Department) وزر لمسح الجدول (Clear Table). ملاحظة اريد التصميم نفس تصميم الصور المرفقة للنوافذ لقد ارفقت لك شاشة تسجيل الدخول وشاشة الرئيسية كبداية باقي الشاشات نعمل عليها خطوة خطوة لكي يكون التصميم مطابق وعندما تنشاء لي الملفات اريد كل ملف باسم النوافذ ليسهل صيانتها او تطويرها لاحقا اسماء الملفات هيه اسماء الخصائص الموجودة في الشريط العلوي كما موضح في الصورة
